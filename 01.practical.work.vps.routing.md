Practical Work 1: VPS Setup & Routing 
=====

## **Install Traceroute**

**For Linux (Ubuntu)**

1. Open Terminal.
2. Install Traceroute via command: *sudo apt-get install traceroute*.
3. Use Traceroute.

## **Check website USTH**
